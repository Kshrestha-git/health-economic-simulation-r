# Health Economic Simulation Model (R)

## Objective
To simulate system dynamics using a Markov state-transition model and evaluate intervention strategies using cost-effectiveness analysis.

## Methods
- Markov state-transition model
- Monte Carlo simulation (n = 1000)
- Cost-effectiveness analysis (ICER)
- Scenario-based intervention modelling

## Key Outputs
- Cost and effect distributions
- Incremental cost-effectiveness ratio (ICER)
- Cost-effectiveness plane

## Tools
- R
- tidyverse

## How to Run
1. Open scripts/simulation_model.R
2. Run the script in RStudio
3. Outputs stored in /outputs
